# Description

Display a breakdown of languages across a GitHub user's repositories.

# Example

```
$ node cli <token> quezlatch
JavaScript..........86.931%
F#..................05.486%
C#..................03.853%
HTML................01.939%
CSS.................01.217%
Ruby................00.280%
Shell...............00.255%
Batchfile...........00.040%
```
