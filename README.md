# Description

Display a breakdown of languages across a GitHub user's repositories.

# Example

## Command line interface

```
$ node cli <token> quezlatch
#repos: 10
JavaScript..........87.477%
F#..................05.520%
C#..................03.611%
HTML................01.951%
CSS.................01.225%
Shell...............00.175%
Batchfile...........00.040%
```
