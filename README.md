# Description

Display a breakdown of languages across a GitHub user's repositories.

# Example

```
$ node index <auth token> quezlatch
JavaScript..........86.931%
F#..................5.486%
C#..................3.853%
HTML................1.939%
CSS.................1.217%
Ruby................0.280%
Shell...............0.255%
Batchfile...........0.040%
```
